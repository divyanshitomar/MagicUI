ALERT

Alert messages can be used to notify the user about something special: danger, success, information or warning.

Alert Examples

There are various type of alerts & want to add this alert in your html you need to add in each type of alert class Name alert-box and then add type of class name given below after each alert listsuccess-alert,warning-alert,error-alert,primary-alert.

Success! Indicates a successful or positive action.

Warning! Indicates a warning that might need attention.

Danger! Indicates a dangerous or potentially negative action.

Info! Indicates a neutral informative change or action.
